Django Template Engine

Blocks

A block ({% verbatim %}{% block name %}{% endblock %}{% endverbatim %}) is an area that a child template can replace or extend. it's a the foundation of the template inheritance mechanism..

For loop

You can iterate over any list passed to the context

{% verbatim %}{% for item in my_list %}
        
  • {{item}}
    • 
        {% endfor %}{% endverbatim %}
      {% for fruit in fruits %}
    • {{ fruit }}
      • {% endfor %}

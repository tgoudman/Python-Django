Presentation of Django

What is Django ? Dango is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It handles much of the complexity of web development, allowing developers to focus on writing their application.

History Django was created in 2003 by Adrian Holovaty and Simon Willison while working at the Lawrence Journal-World newspaper in Kansas. It was released publicy under a BSD licence in July 2005. THe Framework was named after jazz guitarist Django Reinhardt.
In 2008, the Django Software Foundation (DSF) was created to maintain the project. Today Django is one of the popular Python web frameworks, used by companies, Pinterest, and Mozilla.

Key features

  • Follows the MVT (Model-View-Template) pattern
  • Includes a powerful ORM for database management
  • Built-in admin interface
  • Strong security features out of the box
  • Large and active community
